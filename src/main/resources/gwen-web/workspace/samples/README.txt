Gwen workspace
==============

samples
-------

This directory includes sample features. You can keep or delete this directory if you don't want it.

To execute all features in this directory, open a command prompt in the gwen-workspace 
folder and execute the following command:

Linux/Mac:
  ./gwen -b samples

Windows:
  gwen -b samples

Open the gwen-workspace/target/reports/index.html file in a browser to view the 
generated evaluation report.
